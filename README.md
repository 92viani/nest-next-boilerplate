NestJs + Nextjs Boilerplate

## Description

This repository we create a sample project for work nestjs and nextjs together
NestJs handel api and bussines
NextJs handel UI

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
